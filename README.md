# 🎓 Student Management System (Python CLI Project)

## 📌 Description

This is a simple command-line based Student Management System built using Python.
It allows users to add, view, search, and delete student records.

---

## 🚀 Features

* Add new student
* View all students
* Search student by name
* Delete student
* User-friendly CLI interface

---

## 🛠️ Technologies Used

* Python
* Concepts: List, Dictionary, Loops, Conditionals

---

## ▶️ How to Run

1. Install Python
2. Download or clone this repository
3. Run the file:

```
python main.py
```
